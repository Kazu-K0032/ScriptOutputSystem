# ScriptOutputSystem

[English](./docs/lang/en.md) | 日本語

Google Apps Script（GAS）を使ってあにまんchのスレッドスプレッドシートに出力するシステムです。

※YMM4のキャラクターボイスの名前とキャラクター名が一致している必要があります。
※Googleスプレッドシートをclaspでリポジトリ管理しており、ローカル開発環境からGASへのコード反映や、バージョン管理が可能です。
